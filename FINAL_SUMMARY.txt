## 🚀 QUICK START (3 STEPS - 5-10 MINUTES)

### Step 1: Database (2 minutes)
```bash
mysql -u root -p root < quiz-backend/src/main/resources/init-db.sql
```

### Step 2: Backend (1-2 minutes)
```bash
cd quiz-backend
mvn spring-boot:run
```
Backend starts at: **http://localhost:8080**

### Step 3: Frontend (1-2 minutes)
```bash
cd quiz-frontend
npm install  # First time only
npm start
```
Frontend opens at: **http://localhost:3000**
